# AR (Active Response) Nursing Care Portal

Professional Home Healthcare & Medical Equipment Management System.

## 🚀 Project Overview
This platform provides 24/7 ICU Nursing services and medical equipment on rent/sale specifically for the Rawalpindi & Islamabad region. Managed and Founded by **Awais Babar**.

## 🛠 Tech Stack
- **React.js** (Frontend Framework)
- **Tailwind CSS** (Premium Styling)
- **Framer Motion** (Smooth Animations)
- **Lucide Icons** (Clinical Iconography)
- **Vite** (Build Tool)

## 📁 Key Features
- **Dynamic Inventory**: Real-time equipment availability.
- **Service Catalog**: 10+ professional medical services.
- **Admin Dashboard**: Comprehensive order and system health tracking.
- **Mobile Optimized**: Sticky CTA buttons for immediate emergency calls.
- **WhatsApp Integration**: Instant inquiry bridge for patients.

## 🔧 Setup & Installation
1. `npm install` - Install dependencies.
2. `npm run dev` - Start local development server.
3. `npm run build` - Generate production build.
